API Fun!

1) The API I used is one that checks whether or not a phone number is valid. This API is a fast, lightweight, modern, and RESTful JSON API for determining the validity and other details of phone numbers from over 190 countries.
2) The URL of the website: https://app.abstractapi.com/api/phone-validation/documentation.
3) The API is accessible with a GET request and returns the data in JSON format.
4) Applications: mitigate ecommerce fraud by detecting phone numbers associated with malicious behaviour; Validation forms when a user registers their contact information anywhere.
